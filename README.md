# Damir Gumerov | Frontend Developer Portfolio

Персональный сайт-портфолио, разработанный для демонстрации моих навыков, опыта и проектов.

## Стек технологий

| Категория | Технологии |
|-----------|------------|
| **Framework** | Next.js 15 (App Router) |
| **Language** | TypeScript, JavaScript |
| **Styling** | SCSS Modules, CSS Variables |
| **Animations** | Framer Motion |
| **Icons** | Lucide React |
| **Fonts** | Next/Font (Playfair Display, JetBrains Mono) |
| **Analytics** | Vercel Analytics |
| **Deployment** | Vercel |

## Особенности

- 📱 **Полная адаптивность**: Корректное отображение на мобильных, планшетах и десктопах.
- 🎨 **Минималистичный UI**: Акцент на типографике и удобстве чтения.
- 🌍 **Мультиязычность**: Поддержка RU/EN (через Context API).
- 🔗 **Social Previews**: Корректные превью-карточки для Telegram, WhatsApp и VK (OpenGraph).
